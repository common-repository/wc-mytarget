== Description ==

This plugin helps setting up myTarget counter for dynamic remarketing for WooCommerce. 

The plugin requires WooCommerce installed and activated!

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the entire `wc-mytarget` folder to the `/wp-content/plugins/` directory.
1. Activate the plugin through the `Plugins` screen in WordPress

== Frequently Asked Questions ==

= How to setting up myTarget counter? =

1. Open the plugin settings
1. Fill in the fields: myTarget ID and Feed ID from your myTarget account.

= What plugin online store supported by your plugin? =

Only Woocommerce (v3.2.0 or higher).
